# Nightvision
A CS2 nightvision plugin.

It lets each player brighten their screen so dark maps and shadowy corners are easier to see. Players turn it on or off whenever they like and pick how bright it gets, and only the player who turned it on sees the effect - it doesn't change the view for anyone else.

If you have `Clientprefs` installed, each player's on/off choice and brightness level are remembered and reload automatically the next time they join. Without it, the plugin still works, but settings reset when the player leaves.

# Dependencies
For saving player data between sessions, install [Clientprefs](https://github.com/M-archand/Clientprefs-CSS)

Without `Clientprefs`, Nightvision still works, but player settings stay session only.

# How to use
`!nv` to toggle nightvision on or off

`!nvi [#]` to change brightness - pick a number from 1 (dim) to 10 (brightest), default is 5
