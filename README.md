# Nightvision
A CS2 nightvision plugin

# Dependencies
For saving player data between sessions, install [Clientprefs](https://github.com/M-archand/Clientprefs-CSS)

Without `Clientprefs`, Nightvision still works, but player settings stay session-only.

# How to use
`!nv` to toggle

`!nvi [#]` to change intensity
